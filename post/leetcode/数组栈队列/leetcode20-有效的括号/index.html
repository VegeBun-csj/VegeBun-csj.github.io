题目描述

给定一个只包括 ‘('，')'，'{'，'}'，'['，']’ 的字符串，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。

左括号必须以正确的顺序闭合。

注意空字符串可被认为是有效字符串。

题目链接

https://leetcode-cn.com/problems/valid-parentheses

思路

栈+hash表

一般遇到括号的问题都是采用栈来进行解决；遍历字符串

  • 当遍历遇到所有左括号时入栈
  • 当遍历遇到右括号的时候（首先判断栈是否为空，因为如果这个字符串为”}"，那么首先就是不符合，直接返回false）
    • 弹出栈顶元素，然后去hash表中查找是否当前弹出的元素与该右括号在hash表中对应的字符是否相等，如果不相等那么返回false。
    • 如果遍历完字符串后，栈里面还有元素，那么说明存在还没有匹配的左括号，返回false，否则就返回true，表示正好全部匹配

class Solution {
public:
    bool isValid(string s) {
        stack<char> stk;
        unordered_map<char,char> strMap;
        strMap[')'] = '(';
        strMap[']'] = '[';
        strMap['}'] = '{';
        for(int i = 0;i < s.size();i ++){
            if(s[i] == '(' || s[i] == '[' || s[i] == '{') stk.push(s[i]);
     N       else if(s[i] == ')' || s[i] == ']' || s[i] == '}'){
                if(stk.empty()) return false;
                int tempstr = stk.top();
                stk.pop();
                if(tempstr != strMap[s[i]]) return false;
            }
        }
        if(!stk.empty())  return false;
        return true;
    }
};
