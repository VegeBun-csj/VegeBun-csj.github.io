题目描述

给定一个字符串，请你找出其中不含有重复字符的 最长子串 的长度。

题目链接

https://leetcode-cn.com/problems/longest-substring-without-repeating-characters/

思路

hash表 + 双指针

  • hash表用来存放字符串s中的字符；设置两个指针i和j，i 用来遍历字符串的字符，j 一开始再字符串的第一个字符，

  • i每遍历一个字符，就将这个字符作为键存入hash表，值为这个字符出现的次数，同时记录i和j之间的距离(即连续不重复字符的字串的长度)

  • 如果遍历的时候，某个字符在hash表中的值大于1，就说明这个字符重复出现了，此时就出现了重复字符的字串，i 停止向后遍历,然后将j指向的字符在hash 表中的值 –，同时将 j 向后移动，直到当前 i 指向的字符在hash表中的值不大于1为止，然后i再继续向后移动遍历

  • 最终返回最大的连续不重复子串的长度

class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        int res = 0;
        unordered_map<char,int> s_map;
        for(int i = 0,j = 0;i < s.size();i ++){
            s_map[s[i]] ++;
            while(s_map[s[i]] > 1){
                s_map[s[j]] --;
                j ++;
            }

            res = max(res,i - j + 1);

        }
        return res;
    }
};

复杂度分析

时间复杂度为O(N)，空间复杂度为O(N)
