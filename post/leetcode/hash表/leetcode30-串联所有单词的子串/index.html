题目描述

给定一个字符串 s 和一些长度相同的单词 words。找出 s 中恰好可以由 words 中所有单词串联形成的子串的起始位置。

注意子串要与 words 中的单词完全匹配，中间不能有其他字符，但不需要考虑 words 中单词串联的顺序。

示例 1：

输入： s = “barfoothefoobarman”, words = [“foo”,“bar”]

输出：[0,9] 解释： 从索引 0 和 9 开始的子串分别是 “barfoo” 和 “foobar” 。 输出的顺序不重要, [9,0] 也是有效答案。

示例 2：

输入： s = “wordgoodgoodgoodbestword”, words = [“word”,“good”,“best”,“word”]

输出：[]

题目链接

https://leetcode-cn.com/problems/substring-with-concatenation-of-all-words

思路

hash表 + 滑动窗口

题目中说不考虑words中单词的顺序，如果words中有n个单词，那么说明其中的单词可以有n!种顺序，如果要单词数比较少，挨个考虑还行，但是如果单词数目比较多，考虑这么多顺序无疑是不可能的，所以就引入了hash表来解决words中的单词是否与子串中的字符匹配；然后通过滑动窗口来对子串中的每个窗口的单词进行判断是否符合。

  • 为words数组建立一个hash表，来存放words中单词出现的次数；为子串s建立一个hash表，来存放一段窗口中单词出现的次数；如果两个hash表中单词的出现次数都是一样的，那么这个窗口中的单词就是匹配的，就记录下这个窗口的起始位置。

  • 滑动窗口的大小 = words数组中的单词数量*一个单词的长度(每个单词的长度都是一样的)，这样从子串的起始位置开始设置一个上述大小的窗口，窗口滑动的步长为 1

  • 如何对窗口中的单词进行匹配？

    ⚡ 首先确定好窗口之后，对窗口以单词大小进行遍历，对每个遍历到的单词

    1. 首先判断窗口hash表中该单词是否在words的hash表中出现过，如果都没出现过就说明当前窗口不符合匹配原则，直接进入下一个窗口；如果出现过，就将该单词在窗口hash表中的次数+1

    2. 根据两个hash表中的单词次数比较判断窗口中的单词是否匹配。

      • 如果对于窗口中所有的单词，两个hash表中出现的次数都是一样的，就说明完全匹配。将窗口进行滑动，判断下一个窗口是否符合
      • 如果窗口的hash表中单词的出现次数大于words数组的hash表中的次数，就说明不匹配，然后直接将窗口进行滑动，判断下一个窗口是否符合（只要窗口中有一个单词不符合就将进入下一个窗口！！！！！！）

class Solution {
public:
    vector<int> findSubstring(string s, vector<string>& words) {
        vector<int> res;
        if(s.size() == 0 || words.size() == 0) return res;

        unordered_map<string,int> words_map;
        for(auto w : words) words_map[w] ++;

        unordered_map<string,int> s_map;
        //计算出单词组合的总长度
        int len = words[0].size() * words.size();

        //遍历子串,起始位置是慢慢右移的
        for(int start = 0;start + len <= s.size();start ++){
            //每次在新的起点需要清空map
            s_map.clear();
            int j;
            //在子串s中len长的字符串中的所有单词，看是否符合words_map中的条件
            for(j = 0;j < len;j += words[0].size()){
                string tempstr = s.substr(start + j,words[0].size());
                if(words_map[tempstr]){
                    s_map[tempstr] ++;
                    if(s_map[tempstr] > words_map[tempstr]){
                        break;
                    }
                }else{
                    break;
                }
            }
            if(j == len) res.push_back(start);
        }
        return res;
    }
};

复杂度分析

时间复杂度为O(n*m)，空间复杂度为O(N)

n为子串长度，m为words数组字符长度
