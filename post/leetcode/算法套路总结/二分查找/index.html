二分法

对于二分法非常容易出问题的就是查找的边界问题

下面是两个常用的模板框架：

int l = 0;
int r = nums.size() - 1;

while (l <= r) {
    int mid = (l + r) >> 1
    if(nums[mid] == target) return 合适的值
    if(nums[mid] < target) l = mid + 1
    if(nums[mid] > target]) r = mid - 1
}
// 看具体题意，此时 l == r + 1
return l

int l = 0;
int r = nums.size() - 1;
while(l < r){
    int mid = (l + r) >> 1;
    if(nums[mid] <= target) r = mid;
    else l = mid + 1;
}
//此时 l == r
return l;

注意，这个框架下，最后返回的l == r，如果遇到返回的结果可能取到r + 1的情况，需要将一开始的r的值取为nums.size()，而模板一不用加是应为最后返回的l = r + 1，不需要考虑这个问题

例子：https://leetcode-cn.com/problems/search-insert-position/

比如说这一题，在用第二个模板的时候，需要将 r 初始化为 nums.size()，如果设置为为nums.size() -1 就会报错，因为这一题有一个测试用例就是在最后位置插入，而最后位置就是 r + 1

class Solution {
public:
    int searchInsert(vector<int>& nums, int target) {
        int l = 0,r = nums.size() - 1;
        while(l <= r){
            int mid = (l + r) >> 1;
            if(nums[mid] == target) return mid;
            else if(nums[mid] > target) r = mid - 1;
            else l = mid + 1;
        }
        return l;
    }
};

class Solution {
public:
    int searchInsert(vector<int>& nums, int target) {
        int l = 0, r = nums.size();
        while(l < r){
            int mid = (l + r) >> 1;
            if (nums[mid] >= target) r = mid;
            else l = mid + 1;
        }
        return l;
    }
};

注意事项

两个模板地区别:

  • 注意 r 的初始化条件，一般而言都是 nums.size() - 1，但是如果要考虑最后一个插入位置的话，对于模板二需要改为 nums.size()

  • while循环的控制条件，模板一为 l <= r ，模板二为 l < r

  • 模板一中将三种情况分开讨论；

    模板二将==的情况合并，对于上面这种如果写为 nums[mid] >= target，那么就必须写成 r = mid，l = mid + 1

  • 模板一在循环退出的时候，l = r + 1；模板二循环退出的时候为 l == r

例题：

  • 在排序数组中查找元素的第一个和最后一个位置 ：https://leetcode-cn.com/problems/find-first-and-last-position-of-element-in-sorted-array
