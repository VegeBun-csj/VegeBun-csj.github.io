BFS模板

BFS又称宽度优先遍历，类似于树中的层次遍历，是一种从上至下，一层一层进行的遍历，通常采用队列来实现

 void leyerVisit(TreeNode* root) {
        queue<TreeNode*> q;
        TreeNode* cur_front = new TreeNode(0);
        q.push(root);	//根节点入队
        while(! q.empty()){
            cur_front = q.front();	//记录队首节点，因为后面队首出栈后需要将其左右孩子入队
            q.pop();	//队首元素出队
            if(cur_front -> left != nullptr){	//如果存在左孩子则入队
                q.push(cur_front -> left);
            }
            if(cur_front -> right != nullptr){	//如果存在右孩子则入队
                q.push(cur_front -> right);
            }
        }
    }

如果在BFS遍历过程中还需要有其他的操作，只需要在上述的框架中添砖加瓦即可
