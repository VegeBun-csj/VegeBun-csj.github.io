题目描述

给定一个不含重复元素的整数数组。一个以此数组构建的最大二叉树定义如下：

二叉树的根是数组中的最大元素。 左子树是通过数组中最大值左边部分构造出的最大二叉树。 右子树是通过数组中最大值右边部分构造出的最大二叉树。 通过给定的数组构建最大二叉树，并且输出这个树的根节点。

题目链接

https://leetcode-cn.com/problems/maximum-binary-tree

思路

根据本题的题意，我们函数的目的是：找最大的元素作为树的根，然后根据这个最大值元素进行左右划分，然后再找左右区间中的最大值分别作为根的左右孩子。所以明确了目的，我们接下来就是寻找数组中的最大值，然后根据它的索引，对左右子树进行区间的划分，然后递归左右区间寻找其中的最大值即可。

class Solution {
public:

    TreeNode* build(vector<int>& nums,int l,int r){
        if(l > r) return nullptr;
        
        int maxValue = INT_MIN,index = -1;
  		//寻找数组中的最大值
        for(int i = l;i <= r;i ++){
            if(nums[i] > maxValue){
                maxValue = nums[i];
                index = i;
            }
        }
        TreeNode* root = new TreeNode(maxValue);

        //递归处理左右区间，找到根的左右子树的根节点
        root -> left = build(nums,l,index-1);
        root -> right = build(nums,index+1,r);
        return root;
    }


    TreeNode* constructMaximumBinaryTree(vector<int>& nums) {
        //先找出最大值元素和其下标，再以这个下标进行划分，对左右区间进行构造
        return build(nums,0,nums.size() - 1);
    }
};

复杂度分析

时间复杂度为 O(N)，空间复杂度为O(logN)
