题目描述

给定一个二叉树，在树的最后一行找到最左边的值。（这里说的是最后一层的最左边的值，并不是直观上的最左下角的节点哦）

题目链接

https://leetcode-cn.com/problems/find-bottom-left-tree-value/

思路一(BFS)

BFS：通常我们写的BFS是从上往下，从左往右遍历，这样遍历，最后一个遍历的节点为最右下角的节点；但是本题需要求最左下角的节点，所以我们变换一下思路，将BFS的遍历顺序改为从上往下，从右至左遍历，这样到最后一个出队的就是最左下角的节点（思路棒极了，一开始咋没想到呢，还是看了leetcode大佬的解法才知道的，顿时觉得算法真的是太精妙了，有趣😄）

class Solution {
public:
    int findBottomLeftValue(TreeNode* root) {
        //层次遍历(绝妙的思路，从右往左入队列)
        queue<TreeNode*> q;
        TreeNode* cur_front = new TreeNode(0);
        q.push(root);
        while(! q.empty()){
            cur_front = q.front();
            q.pop();
            if(cur_front -> right != nullptr){
                q.push(cur_front -> right);
            }
            if(cur_front -> left != nullptr){
                q.push(cur_front -> left);
            }
        }
        return cur_front -> val;
    }
};

BFS代码(二)

这种写法也是BFS，但是是还是以前的做法，从上往下，从左往右，这里主要是引入了两个指针节点，temp指针用来记录每一层遍历的时候的第一个节点，最后通过它来获取最后一层的最左侧节点；而node虽然也是同样的功能，但是它是用来将左右节点入队列的，是在循环体内的。

class Solution {
public:
    int findBottomLeftValue(TreeNode* root) {
        queue<TreeNode*> q;
        TreeNode* temp = new TreeNode(0);
        q.push(root);
        while(! q.empty()){
            temp = q.front();
            int size = q.size();
            while(size --){
                TreeNode* node = q.front();
                q.pop();
                if(node -> left){
                    q.push(node -> left);
                }
                if(node -> right){
                    q.push(node -> right);
                }
            }
        }
        return temp -> val;
    }
};

思路二

DFS，纵深地遍历，用一个变量deep_max记录树的最大深度，一个变量res记录最左下角的节点注意：我们需要找的是最左下角的节点，所以DFS中应该先遍历左孩子

  • DFS先遍历左孩子就能先在最左下角更新最大深度，同时将最左下角的节点值更新到res
  • 遍历到最左下角的节点刚好在为最大深度，则后面的节点(即使处于最大深度处)也不能更新res的值，这样最终返回的就是最左下节点了；而如果当前节点不在最大深度，也没关系，继续遍历呗，回溯，再深地走，走到最深处再更新res即可

所以说来说去，先遍历左子树就是因为，处于最大深度的节点很多(大家都有能力)，而res这个宝座谁都想坐，那怎么保证最左下节点能稳居宝座呢？那就是先让他上咯，先到先得嘛！如果后面确实还有实力更深的，咱再让位呗，总之咱就得第一个来！！！

  • 图一：最大深度deep_max和res，只会在节点 4，2，1进行更新
  • 图二：最大深度deep_max和res，一开始遍历左子树会在节点 4，2，1进行更新，但是后面节点3的深度更深，所以最终在节点3更新最大深度和res.

class Solution {

private:
    //记录最大的深度和此时的节点值
    int deep_max = -1,res;

public:

    void DFS(TreeNode* root,int deep){
        if(root == nullptr) return;
        //当遍历到最大深度后，后面比这个深度小的层都不会执行更新
        if(deep_max < deep){
            deep_max = deep;
            res = root -> val;
        }

        DFS(root -> left,deep + 1);
        DFS(root -> right,deep + 1);
    }

    int findBottomLeftValue(TreeNode* root) {
        DFS(root,0);
        return res;
    }
};

复杂度分析

BFS时间复杂度为O(N)，空间复杂度为O(N)；

DFS时间复杂度为O(n)，空间复杂度为O(logN)，最坏的情况是为满二叉树的时候，空间复杂度为O(N)
