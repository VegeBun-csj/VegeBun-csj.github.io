题目描述

序列化是将一个数据结构或者对象转换为连续的比特位的操作，进而可以将转换后的数据存储在一个文件或者内存中，同时也可以通过网络传输到另一个计算机环境，采取相反方式重构得到原数据。

请设计一个算法来实现二叉树的序列化与反序列化。这里不限定你的序列 / 反序列化算法执行逻辑，你只需要保证一个二叉树可以被序列化为一个字符串并且将这个字符串反序列化为原始的树结构。

请实现两个函数，分别用来序列化和反序列化二叉树。（当然自己序列化的格式不限定）

示例:

你可以将以下二叉树：

1
2
3
4

    1
   / \
  2   3
     / \
    4   5

序列化为 "[1,2,3,null,null,4,5]"

思路

先序遍历

  • 序列化：当根节点为空的时候，读入”#"，直接退出；否则就读入它的值，然后递归遍历左右子树，即可将树的元素都读入ostrinstream,最后将stream流转为string类型
  • 反序列化：首先通过isringstream将数据先存入stream流中，然后一个一个读出来，如果一开始读到字符"null”,就返回空指针(表明是空树)；否则构造等于这个值的根节点，然后递归遍历，生成左右子树

class Codec {
public:
    void serial(TreeNode* root,ostringstream& out){
        if(! root){
            out << "# ";
            return;
        }
        //读入根节点的值，后面用空格分割，这是为了方便方便istringstream以空格进行分割
        out << root -> val << " ";

        serial(root -> left,out);
        serial(root -> right,out);
    }

    TreeNode* deserial(istringstream& in){
        string val;
        in >> val;
        if(val == "#") return nullptr;
        TreeNode* root = new TreeNode(stoi(val));
        root -> left = deserial(in);
        root -> right = deserial(in);
        return root;
    }

    // Encodes a tree to a single string.
    string serialize(TreeNode* root) {
        ostringstream out;
        serial(root,out);
        return out.str();
    }

    // Decodes your encoded data to tree.
    TreeNode* deserialize(string data) {
        istringstream in(data);
        return deserial(in);
    }
};

复杂度分析

序列化：时间复杂度为O(N)，空间复杂度为O(N) 反序列化：时间复杂度为O(N)，空间复杂度为O(N)
