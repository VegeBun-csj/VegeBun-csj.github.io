题目描述

给定一个二叉树，原地将它展开为一个单链表。

例如，给定二叉树

题目链接

https://leetcode-cn.com/problems/flatten-binary-tree-to-linked-list/

思路

二叉树的题目都可以使用递归来做，对于递归，我们并不需要深入到递归函数一层一层分析，只需要关注函数的功能，剩下的就是考虑在当前层应该干嘛，其他的不用管，只管去调用即可。

比如这一题，我们要实现的就是将整棵树变成一个单链表，所以我们定义的函数flatten功能就是将树变为单链表，而如果我们将左子树变为一个单链表，右子树变为一个单链表，然后将左边的单链表接到右边就能将整棵树变为一个单链表。所以：

  • 我们可以先递归左右子树，将它们分别变为一个单链表
  • 然后将这两个单链表还有根节点进行拼接即可

代码注意点：在最后拼接的时候是针对根节点和两个左右子树链表进行操作，注意是将根节点的右指针指向根节点的左子树链表头，将这个链表尾指向右子树链表，这里有一个循环操作，不能写成while，因为左子树链表可能不止一个，需要遍历到最后一个节点再和右子树链表拼接才行。eg：根节点和左右子树拼接的时候，左子树链表为2，3，4，我们需要将右子树链表5，6拼接在4后面，如果使用 if ，那么5就会拼在2后面，然而使用while的时候会找到4这个节点，然后将4指向5，就完成了整个链表的构造。

Tips:

其实这题跟归并排序的思路很像，先递归处理区间，然后进行合并。

class Solution {
public:
    void flatten(TreeNode* root) {
        if(! root) return;
        flatten(root -> left);
        flatten(root -> right);
        
        TreeNode* temp = root -> right;
        root -> right = root -> left;
        root -> left = nullptr;

        //这里使用循环是因为左子树可能有多个节点
        while( root -> right != nullptr) root = root -> right;
        //将右子树的第一个节点连接但左子树的最后一个节点后面
        root -> right = temp;
        return;
    }
};

复杂度分析

时间复杂度为O(N)，空间复杂度为O(logN)
